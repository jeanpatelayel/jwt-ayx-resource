# jwt-ayx-resource

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何单个域名或具体网站，主要作为资源文件的集合与分发点。

## 项目简介

jwt-ayx-resource 是一个轻量级的静态资源仓库，旨在集中存放各类 HTML 页面及相关前端资源。仓库内容以独立、可复用的页面为主，便于快速部署或引用。

## 目录说明

```
jwt-ayx-resource/
├── pages/          # 存放 HTML 页面文件
├── assets/         # 页面所需的样式、脚本、图片等静态资源
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立的 HTML 页面。
- `assets/`：按页面或功能模块组织的公共资源。

## 页面归档说明

- 所有页面均为独立文件，不依赖仓库外部的动态服务。
- 页面内容可能包含示例数据、占位文本或功能演示，不构成实际服务。
- 归档的页面可能不定期更新或清理，请以仓库最新状态为准。

## 维护说明
